# Product

## Register

product

## Users

咸鱼之王玩家和工具维护者在浏览器里管理多个游戏账号 Token，切换角色，执行日常、批量任务、消息测试和网页游戏入口等操作。

## Product Purpose

北冕之王 Web Helper 是一个 Vue Web 工具台，用本地保存的 Token 与游戏服务通信，减少重复操作，并提供可直接进入游戏的网页端入口。成功体验是账号切换清楚、状态反馈及时、游戏画面优先、辅助功能不遮挡主要任务。

## Brand Personality

实用、直接、稳定。界面可以保持游戏工具的轻松感，但核心操作应像管理后台一样可靠、可扫读。

## Anti-references

避免营销页式首页、过大的说明面板、装饰性卡片、弹窗式入口和会遮挡游戏画面的辅助内容。网页游戏模块尤其应保持轻量，只保留进入游戏所需的账号选择与刷新操作。

## Design Principles

- 让主要任务先出现，管理功能只在需要时露出。
- 账号与连接状态必须明确，切换操作要可预期。
- 保持与现有 Naive UI、顶部导航和 Token 管理体验一致。
- 对移动端和窄屏优先保证控件可用、文字不挤压游戏画面。

## Accessibility & Inclusion

遵循基础 WCAG AA 对比度和键盘可操作要求。避免依赖颜色单独表达状态，减少动画和闪烁，控件保持清晰的焦点与禁用状态。
