# 首页登录记住密码设计

## 背景

用户已确认方案 A：登录成功后保存用户名；增加“记住密码”选项；勾选后保存密码，未勾选则只保存用户名并清除已保存密码。

## 范围

- 仅修改首页登录弹窗 `src/views/Home.vue`。
- 不改变系统登录态 `SYSTEM_SESSION_KEY`。
- 不改变注册校验、注册成功逻辑和超级管理员登录逻辑。
- 本地表单偏好独立存储，建议 key 为 `xyzw_system_auth_remember`。

## 交互

- 登录模式显示“记住密码”复选项，注册模式不显示。
- 打开登录弹窗时读取本地保存的用户名和记住密码状态。
- 如果保存状态为记住密码，同时回填密码。
- 登录成功后保存用户名。
- 登录成功且勾选记住密码时保存密码。
- 登录成功但未勾选记住密码时保存用户名和 `rememberPassword: false`，不保存密码字段。

## 视觉

这是现有登录弹窗的小控件增强，不做整体重设计。复选项放在密码输入下方、反馈文案上方，沿用当前 wheat 金色、暗色玻璃表单、0.9rem 圆角体系，避免撑开移动端宽度。

## 验证

- 新增静态契约测试覆盖 UI、读取、保存和登录成功触发保存。
- 运行首页登录相关测试。
- 运行生产构建。
